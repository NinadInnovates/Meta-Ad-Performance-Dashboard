# Meta-Ad-Performance-Dashboard

Project Overview :

This is a Meta Ad Performance Dashboard that tracks the effectiveness of ad campaigns across key KPIs such as impressions, clicks, engagements, conversions, and budget. It provides a complete funnel view—from awareness to engagement to purchases—along with demographic, geographic, and time-based insights.

Objectives:

Analyze ad performance across different campaigns.
Identify high-performing audience segments.
Track engagement and conversion metrics.
Optimize marketing spend for better ROI.

Tools & Technologies:

Power BI – Dashboard creation
Excel/Python – Data cleaning & preprocessing

Key Metrics (KPIs):

Impressions: 216K
Clicks: 25.4K
CTR: 11.76%
Conversion Rate: 5.21%
Purchase Rate: 0.61%
Total Budget: $2.54M

Dashboard Features:

📊 Campaign performance overview
👥 Audience analysis (Age, Gender)
🌍 Geographic performance insights
⏱️ Time-based trends (Hourly & Weekly)
🎯 Ad type comparison (Stories, Carousel, Image, Video)

Key Insights:

Top-performing campaigns generated the highest CTR and conversions.
Majority of clicks came from the 25–34 age group.
Video ads showed higher engagement but lower conversion rate.
Certain regions contributed significantly to overall revenue.

Business Recommendations:

Increase budget for high-performing campaigns.
Focus on audience segments with high engagement.
Optimize underperforming ad creatives.
Invest more in channels with better ROI.

Future Improvements :

Add predictive analysis (ML model).
Automate data pipeline.
Integrate real-time data.
